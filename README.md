# React-Product-Explorer
 React application that consumes an external product API, allowing us to display products and filter them by name, category, and price.


### Features
- Fetch users from external API
- Search products by name (debounced), category, and price
- View product details in a controlled modal
- Smooth CSS animations
- Clean component architecture

### Tech Stack
- React
- Vite
- JavaScript (ES6+)
- CSS


### Concepts Applied
- Custom Hooks
- State lifting
- Conditional rendering
- Controlled components
- Separation of concerns
- useMemo to avoid the recalculation of the filter
- Skeleton loader
- Persistence of query params filters
